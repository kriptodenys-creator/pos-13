# Мобільний інтерфейс приходу товару

## Проблема

На планшеті незручно робити приход товару через основний інтерфейс складу - багато полів, дрібні кнопки, потрібно багато прокручувати.

## Рішення

Створено спрощений мобільний інтерфейс `/admin/inventory-mobile` оптимізований для планшетів.

## Особливості

### ✅ Великі елементи інтерфейсу
- Великі кнопки (мінімум 48x48px для зручного натискання пальцем)
- Великі поля вводу (висота 48-64px)
- Великий шрифт (16-24px)

### ✅ Швидке додавання
- **Кнопки швидкого додавання**: +1, +5, +10, +20, +50, +100
- Одним натисканням додаєте потрібну кількість

### ✅ Калькулятор упаковок
```
Упаковок: 5
Одиниць в упаковці: 10
= Всього: 50 кг
```
Зручно для приходу товару в упаковках/коробках

### ✅ Мінімум полів
Тільки необхідні поля:
- Кількість (з швидкими кнопками)
- Калькулятор упаковок
- Ціна за одиницю (автоматично підставляється)
- Причина (автоматично "Приход товару")

### ✅ Пошук
- Великий рядок пошуку вгорі
- Пошук по українській та литовській назві
- Миттєвий результат

### ✅ Візуальний список
- Картки товарів з фото
- Великі картки для зручного натискання
- Показ поточного залишку

## Як використовувати

1. **Відкрийте на планшеті**: `/admin/inventory-mobile`
2. **Знайдіть товар**: Введіть назву в пошук або прокрутіть список
3. **Натисніть на товар**: Відкриється форма приходу
4. **Додайте кількість**:
   - Використайте швидкі кнопки (+5, +10, +50)
   - АБО введіть вручну
   - АБО використайте калькулятор упаковок
5. **Підтвердіть**: Натисніть зелену кнопку "Підтвердити приход"

## Приклад роботи

### Приход курки в упаковках
```
1. Знайти "Курка"
2. Натиснути на картку
3. Калькулятор упаковок:
   - Упаковок: 3
   - Одиниць в упаковці: 5.6 (кг в одній упаковці)
   - = Всього: 16.8 кг
4. Ціна: 3.50 € (автоматично)
5. Підтвердити
```

### Швидкий приход овочів
```
1. Знайти "Помідори"
2. Натиснути на картку
3. Швидкі кнопки: +10, +10, +5 = 25 кг
4. Підтвердити
```

## Доступ

- **URL**: `/admin/inventory-mobile`
- **З головного меню**: Панель адміністратора → "Приход товару (планшет)"
- **Захист**: Потрібен PIN-код адміністратора

## Переваги vs основний інтерфейс

| Функція | Основний | Мобільний |
|---------|----------|-----------|
| Розмір кнопок | Маленькі | Великі (48-64px) |
| Швидке додавання | Ні | Так (+1, +5, +10...) |
| Калькулятор упаковок | Так | Так (спрощений) |
| Редагування товару | Так | Ні |
| Історія руху | Так | Ні |
| Списання | Так | Ні |
| Приход | Так | **Оптимізовано** |

## Технічні деталі
