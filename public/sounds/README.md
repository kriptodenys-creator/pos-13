# Звукові файли для POS системи

## Інструкція по додаванню MP3 файлу для таймера фритюра

Щоб додати власний звуковий сигнал для таймера фритюра:

1. **Підготуйте MP3 файл:**
   - Назва файлу: `timer-complete.mp3`
   - Рекомендована тривалість: 1-3 секунди
   - Рекомендована гучність: нормалізована
   - Формат: MP3

2. **Розмістіть файл:**
   - Помістіть файл `timer-complete.mp3` в цю папку: `/public/sounds/`
   - Повний шлях: `c:/Users/User/Desktop/pos/public/sounds/timer-complete.mp3`

3. **Перезавантажте сторінку:**
   - Оновіть сторінку фритюра в браузері
   - Система автоматично використає ваш MP3 файл

## Альтернатива

Якщо MP3 файл не знайдено, система автоматично використовує синтезований звук (3 звукові сигнали).

## Рекомендовані звуки

Ви можете використати:
- Звук таймера кухні
- Звук дзвінка
- Звук будильника
- Будь-який короткий помітний звук

## Приклад використання

```javascript
// Код автоматично намагається завантажити:
const audio = new Audio('/sounds/timer-complete.mp3')
audio.volume = 0.8
await audio.play()
```

---

**Примітка:** Файл має бути в форматі MP3 для кращої сумісності з браузерами.
