# 🚀 Швидкий старт POS системи

## ⚡ Найшвидший спосіб запуску

### Варіант 1: Development режим (рекомендовано для розробки)
```bash
start-fast.bat
```
**Час запуску:** ~15-25 секунд

### Варіант 2: Production режим (НАЙШВИДШИЙ)
```bash
start-production.bat
```
**Час запуску:** ~5-10 секунд (після першої збірки)

### Варіант 3: Перезапуск
```bash
restart.bat
```

## 💡 Додаткові поради

### Якщо все ще повільно:

1. **Додайте папку в виключення антивірусу**
   - Windows Security → Exclusions
   - Додайте `C:\Users\User\Desktop\pos12`

2. **Перезавантажте комп'ютер**
   - Очистить пам'ять та процеси

3. **Закрийте інші програми**
   - Браузери з багатьма вкладками
   - Важкі додатки

4. **Очистіть все та переустановіть** (якщо нічого не допомагає)
   ```bash
   rmdir /s /q node_modules
   rmdir /s /q .next
   npm install
   ```
